
# What is Virtulisation ?
=========================

Virtualization in computer science often refers to the abstraction of some physical 
component into a logical object.

# What is virtual machine ?

A virtual machine (VM) can virtualize all of the hardware resources, including processors, memory, storage, 
and network connectivity.

# What is Hypervisor ?

Hypervisor is the software that provides the environment in which the VMs operate.

- Just like, Operating System schedule threads of different processes. Hypervisor schedules
different/same OS system requestes with minimal letancy.

- Just like, Operating system provide isolation between several thread between Hypervisor do the same.
